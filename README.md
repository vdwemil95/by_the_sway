# By the sway
My personal desktop setup full of Wayland goodness

Hotkeys for sway and applications are based on Emacs hotkeys.

## Requirements
  * Rofi (Need to switch to wofi for peak Wayland)
  * Waybar
  * Mako
  * Alacritty (terminal)
  * Sway

## Spotify hotkeys
  * Super+Ctrl+j Play/Pause
  * Super+Ctrl+n Next
  * Super+Ctrl+p Previous
  
## Wofi hotkeys
  * Ctrl+p up
  * Ctrl+n Down
  * Ctrl+g Exit
